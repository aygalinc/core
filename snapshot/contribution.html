Contribute

The chameleon-core is an open-source project and all contributions are welcome to assist with its development and maintenance.

The Chameleon Core project is part of the OW2 Chameleon project, and is licensed under the Apache License 2.0.

Issues (bug and feature tracker)

Please report any bugs found, feature requests or other issues on Chameleon Core GitHub tracker.

Fork, patch and contribute code

Support

Support is done using the OW2 Chameleon mailing lists. Check the mailing list page.
